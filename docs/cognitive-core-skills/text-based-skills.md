# Text-Based Cognitive Skills

Text-based cognitive skills are language-first capabilities used for analysis, synthesis, explanation, transformation, and machine-readable artifact production.

## Scope

- Reading comprehension
- Instruction following
- Summarization
- Question answering
- Information extraction
- Entity extraction
- Claim extraction
- Citation handling
- Source comparison
- Contradiction detection
- Taxonomy and ontology creation
- Classification and labeling
- Translation and tone transformation
- Drafting and report writing
- Policy interpretation
- Requirements analysis
- Code understanding and generation
- Code review
- Data-to-text and text-to-structured-data conversion
- Structured output generation
- Wiki, index, and log maintenance

## Text skill quality dimensions

| Dimension | Operational meaning |
|---|---|
| Fidelity | Preserves source meaning without unsupported additions |
| Completeness | Covers required information and constraints |
| Structure | Matches expected format and schema |
| Traceability | Links claims to evidence and citations |
| Utility | Produces output usable by downstream users or systems |

## LLM and SLM positioning

- LLMs are strongest in broad synthesis, cross-domain transfer, and nuanced drafting.
- SLMs are strongest in constrained routing, deterministic extraction, low-latency checks, and local/private deployment.

SLM-friendly text patterns:

- Intent routing
- Classification
- Extraction
- Policy checks
- Template filling
- Lightweight transformation
