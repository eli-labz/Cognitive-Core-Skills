# Glossary

## Key terms

- Cognitive core skills: operating capabilities that enable useful, bounded, goal-driven AI work.
- LLM: large language model optimized for broad language generation and reasoning.
- SLM: small language model optimized for constrained, low-latency, private, or edge tasks.
- AI Agent: orchestration layer combining model, memory, planning, tools, verification, and action control.
- World model: representation of environment state and transition dynamics.
- Grounding: linking outputs and decisions to external source-of-truth evidence.
- Verification: post-action confirmation and mismatch detection.
- Escalation: transfer of control to human oversight under policy-defined conditions.
- Cognitive debt: risk accumulated when human understanding and accountability are displaced.
- Skill card: machine-readable record for one cognitive capability.
- Related skill: referenced capability with functional dependency or complementarity.
- Evidence ledger: trace map connecting claims to sources over time.
- Provenance: origin and transformation record for knowledge claims.
- Data lineage: upstream and downstream dependency chain of data assets.
- OKF: open knowledge format style principles for interoperable catalog artifacts.
- LLM Wiki: persistent, compounding markdown knowledge system maintained by model-driven workflows.
