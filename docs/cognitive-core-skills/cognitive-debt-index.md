# Cognitive Debt Index (CDI)

Cognitive Debt Index (CDI) measures how much human judgment, skill retention, task understanding, and accountability are displaced by AI without adequate review or learning.

## CDI dimensions

- Decision override rate
- Human cognitive retention
- Strategic engagement
- Escalation quality
- Verification participation
- Blind acceptance rate
- Skill decay risk
- Over-automation risk
- Accountability clarity
- Human learning loop strength

## Scale (0 to 100)

| Score band | Interpretation |
|---|---|
| 0-20 | Healthy augmentation |
| 21-40 | Mild cognitive debt |
| 41-60 | Moderate cognitive debt |
| 61-80 | High cognitive debt |
| 81-100 | Critical cognitive dependency |

## Calculation guidance

- Score each dimension from 0 to 10.
- Normalize to 0-100 by summing weighted dimensions.
- Recommended initial weighting: equal weights.
- Calibrate quarterly with observed performance and incident patterns.

## Mitigation controls

- Require human explanation before accepting high-impact outputs.
- Rotate manual review duties.
- Preserve training tasks in each workflow.
- Use verification checklists.
- Track override decisions and rationale.
- Require source inspection for critical claims.
- Maintain human skill maps.
- Use escalation gates for high-risk decisions.

## Governance use

- Track CDI by team, workflow, and quarter.
- Link CDI trends to incident and quality outcomes.
- Trigger intervention plans when CDI enters high-risk bands.
