# Roadmap

## Vision

Make Cognitive Core Skills the default open taxonomy for evaluating and designing production-grade AI workers.

## Near-term

- Add schema-level duplicate-name safeguards in CI.
- Add benchmark expansion by domain with trace-backed fixtures.
- Add release automation for taxonomy version increments.
- Add contributor stats and changelog automation.

## Mid-term

- Add richer evaluator packs mapped to rubric dimensions.
- Add migration guides for downstream adopters.
- Add normalized skill relationship typing and visualization.
- Add multilingual taxonomy descriptions.

## Long-term

- Publish ecosystem benchmark leaderboard format.
- Publish reference implementation examples per domain.
- Establish governance council for taxonomy evolution.

## Community asks

- Propose missing skills via issue templates.
- Share task traces to strengthen benchmark coverage.
- Open PRs with narrowly scoped, test-backed improvements.
